# Game Dashboard

A Chrome extension that renders live game data by fetching from public APIs and displaying it in a clean dashboard popup.

## Features
- Real-time data fetching via REST APIs
- Clean popup UI with stats display
- Lightweight — no external dependencies

## Installation
1. Open Chrome and go to `chrome://extensions/`
2. Enable Developer Mode
3. Click "Load unpacked" and select this folder

## Tech Stack
- JavaScript (Chrome Extensions API)
- HTML/CSS
